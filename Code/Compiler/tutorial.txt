Variable Types: 
    int, string, bool

Variable Initialization Syntax: 
    Type "Var" (then space, this is importaint) followed by a name starting with anything that isnt a number and/or opperator then "=" whatever value

    Int: Var name = a number
    String: Var name = "String Name"
    Bool: Var name = True or False

Math Orithmitics: Addition, Subtraction, Division, Multiplication 
    Addition Syntax: add(Var/Number + Var/Number + ... + Var/Number)
    Subtraction Syntax: sub(Var/Number - Var/Number - ... - Var/Number)
    Division Syntax: div(Var/Number \ Var/Number \ ... \ Var/Number)
        NOTE: There will be checkings to make sure nothing gets divided by 0
    Multiplication Syntax: mul(Var/Number * Var/Number * ... * Var/Number)

Comparison Orithmitics:
    Less Than: <(Var/Number, Var/Number)
    Greater Than: >(Var/Number, Var/Number)
    Equals: =(Var/Number, Var/Number)

    And: &(Bool & Bool & ... & Bool)
    Or: |(Bool | Bool | ... | Bool)
    N-And: !&(Bool & Vool & ... & Bool)
    N-Or: !|(Bool | Bool | ... | Bool)

Output:
    OUT(Variable)
